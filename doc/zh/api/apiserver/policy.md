# Policy接口
## 1. 创建Policy
### 1.1 接口描述

### 1.2 接口方法
POST /v1/policy
### 1.3 输入参数
**Body参数**
参数名称 | 必选 | 类型 | 描述

metadata | 是 | ObjectMeta | 元数据
username | 是 | string | 用户名称
policy | 是 | AuthzPolicy | 策略
### 1.4 输出参数
参数名称 | 类型 | 描述
------------ | ------------- | ------------
metadata | ObjectMeta | 元数据
username | string | 用户名称
policy | AuthzPolicy | 策略
