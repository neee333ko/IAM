# 授权接口
## 1. 授权
### 1.1 接口描述
授权。
### 1.2 请求方法
POST /v1/authz
### 1.3 输入参数
**Body参数**
参数名称 | 必选 | 类型 | 描述
------------ | ------------- | ------------ | -------------
resource | 是 | string | 资源
action | 是 | string | 动作
subject | 是 | string | 主体
context | 否 | ladon.Context | 上下文
### 1.4 输出参数
参数名称 | 类型 | 描述
------------ | ------------- | ------------
result | string | 授权结果
error | string | 错误信息
